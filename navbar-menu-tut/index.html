In this Angular 8 tutorial, we'll learn to build an Angular Navbar/Menu using NG Bootstrap The aim of the tutorial is to explore the abilities embedded in the ng bootstrap in making our nav/menu responsive and user friendly for navigational purposes.

Setup Project Environment
$ ng new ng-tuty --skip-install

This command will create the angular project without installing our dependencies

Note: You can choose to run ng new ng-tuty and your dependencies will get installed along the project initialization. Your packages will them be installed using NPM. With the method above I choose to use yarn in this tutorial. 

// Start the project using yarn
$ yarn init
$ yarn install

Running yarn init will initialize our project to use yarn and generate for us yarn.lock file 

$ yarn add bootstrap
$ yarn add @fortawesome/fontawesome-svg-core  @fortawesome/free-solid-svg-icons @fortawesome/angular-fontawesome@0.5.0
$ yarn start

Configure NG Bootstrap and Angular FontAwesome
NG Bootsrap
"styles": [
"node_modules/bootstrap/dist/css/bootstrap.min.css"
]
import { FontAwesomeModule } from '@fortawesome/angular-fontawesome';
